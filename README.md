## Overview

**Team Connect** is a modern, scalable **Room Booking System** built with Flask and Firebase. It enables organizations to intelligently manage conference rooms, track bookings, detect scheduling conflicts, and provide real-time availability insights.

### Key Highlights:
- 🔐 **Google OAuth 2.0 Authentication** via Firebase
- 📊 **Real-time Occupancy Tracking** with occupancy percentage calculation
- ⚡ **Instant Booking Clash Detection** to prevent double bookings
- 👥 **User Statistics Dashboard** tracking booking history
- 📈 **Smart Availability Suggestions** finding earliest free slots
- 🌐 **Cloud-based Infrastructure** using Google Cloud Platform
- 🧪 **Comprehensive Test Suite** with unit and integration tests
