Key features

  • 24-hour booking: reserve rooms at any hour of the day, with no 9-to-6 limits.
  • Create and manage rooms: add new meeting rooms and manage the ones you own.
  • Clash-free scheduling: every booking is checked so meetings never overlap in the same room.
  • Occupancy insights: occupancy figures and free-slot suggestions across the next several days.
  • Profiles and activity stats: personal profile with live counts of rooms and bookings.

Technologies used

Technology Role Description
Flask (Python) Backend and routing Web framework serving pages and JSON APIs for rooms, bookings and the chat endpoint.
Firebase Firestore Database NoSQL document store holding rooms, days, bookings and user profiles.
Firebase Authentication Sign-in Secure Google authentication; each request is tied to a verified user identity.
Jinja2 templates Server rendering HTML templates (base, dashboard, room, profile, about) rendered on the server for initial data.
Plain HTML and JavaScript Frontend Unstyled HTML with the fetch API; all create, read, update and delete actions call JSON endpoints.

How it works

Create a room

  1. Open the Dashboard and use the "Add New Room" field,
  2. Enter a unique room name - duplicate names are rejected automatically.
  3. The room is saved and instantly appears in the room list without a page refresh.
  4. You become the room owner, and it shows under "Rooms I created" on your profile.

Book a room

  1. Pick a room, a date, a start and end time, and give the meeting a name.
  2. Booking is allowed at any time of day - a full 24 hours are available.
  3. The system checks for clashes; overlapping times in the same room are blocked.
  4. On success the booking is stored and appears on the room calendar and occupancy figures.

Reschedule a booking

  1. Go to your Profile or a room, and click Edit on the booking.
  2. Change the date and/or the start and end times, then save.
  3. The new slot is re-checked for clashes before it is accepted.

Cancel a booking

  1. Find the booking on the room page or your dashboard and use the delete option.
  2. The reservation is removed and the room calendar frees up immediately.
